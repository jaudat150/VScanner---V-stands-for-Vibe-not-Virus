# VScanner ( V stands for Vibe not Virus ) (Windows)

A lightweight local security scanner + GUI for inspecting suspicious processes, startup entries, services, and binaries.

Includes:
- Process monitor with heuristics (hidden/suspicious paths/network usage)
- Non-Microsoft services viewer
- SHA256 hashing
- VirusTotal hash lookup
- Copy / Export CSV support

## How to Use
- Put Your virus total API_Key in the config file (or your can put it in the field specified in the app)
- You are good to go
## Requirements
- Windows
- Python 3.8+

## Install
```bash
pip install -r requirements.txt
```
